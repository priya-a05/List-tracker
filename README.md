# List-tracker
# 📝 To-Do List App (Tkinter)
A simple and beginner-friendly To-Do List application built using Python’s **Tkinter** library.
This app helps you keep track of your daily tasks with a clean interface and basic task management features.

## 🚀 Features
* ➕ Add new tasks
* 🗑️ Delete selected tasks
* 🧹 Clear all tasks at once
* 🎯 Set priority levels:
  * 🔴 High (Red)
  * 🟡 Medium (Yellow)
  * 🟢 Low (Green)
* 🖥️ Simple and minimal GUI

## 🛠️ Tech Used
* Python
* Tkinter (built-in GUI library)

## 📂 How to Run

1. Make sure Python is installed (Python 3 recommended)
2. Clone this repository:
bash
get clone https://github.com/priya-a05/List-tracker.git  
4. Navigate to the project folder:
bash
cd List-tracker
6. Run the file:
bash
   python GUI.py
   
## 📌 How It Works
* Enter a task in the input field
* Click **"Save Task"** to add it to the list
* Select a task to:
  * Delete it
  * Set its priority (color-coded)
* Use **"Clear All"** to remove everything

## ⚠️ Limitations
* Tasks are not saved permanently (no database or file storage yet)
* No edit/update feature for tasks
* Basic UI (can be improved with styling)
